JABCode Color Modes Comparison

JABCode (Just Another Barcode) is a color 2D matrix symbology that supports multiple color modes. This page compares the different color modes supported by the Java wrapper.

The native JABCode library directly supports 4 and 8 color modes. For other color modes, the Java wrapper implements color quantization, custom palettes, and dithering techniques.
